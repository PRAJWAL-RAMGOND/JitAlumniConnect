## 🆘 Troubleshooting

### Can't Access Website?
```
1. Check server is running
2. Go to: http://localhost:8000/login.html
3. Clear browser cache
4. Try incognito mode
```

### Admin Login Not Working?
```
1. Use exact credentials:
   Email: admin@jit.ac.in
   Password: admin123
2. Check for typos
3. Clear localStorage
4. Try again
```

### Chatbot Widget Not Showing?
```
1. Login as student (not admin)
2. Go to dashboard or any page
3. Look for pulsing 🤖 icon (bottom-right)
4. Check browser console for errors
```

### Features Not Working?
```
1. Check browser console (F12)
2. Verify JavaScript is enabled
3. Clear cache and reload
4. Check localStorage data
```
