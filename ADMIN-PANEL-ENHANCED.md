## 🎯 Usage Guide

### View Analytics
```
1. Click "Analytics" in sidebar
2. Charts load automatically
3. Change time range with dropdown
4. Export report with button
5. Hover over elements for details
```

### View Activity Logs
```
1. Click "Activity Logs" in sidebar
2. See all recent activities
3. Filter by type or date
4. Export logs as needed
5. Click entries for details
```

### Navigate Dashboard
```
1. Dashboard shows overview
2. Click "View All" on Recent Activity
3. Redirects to full Activity Logs
4. All sections interconnected
```
