## 📋 Profile Page Now Shows

When you login and visit the profile page, you'll see:

1. **Profile Header**
   - Avatar with first letter of your name
   - Full name
   - Role (Student/Faculty/Alumni/Department)
   - Email and role-specific details

2. **Role-Specific Information**
   - **Students**: Branch, Year, USN
   - **Faculty**: Department, Designation, Experience
   - **Alumni**: Batch, Company, Designation
   - **Department**: Department Name, Description

3. **Your Posts Section**
   - All posts you've created
   - Like and comment counts
   - Post images if any
   - Empty state if no posts yet
